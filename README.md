# Galaxy Web OS - Mobile First Homepage

A modern, lightweight, and responsive web app homepage designed for Android WebView and mobile browsers.

## Features
- **Galaxy Theme**: Animated CSS stars and deep space gradients.
- **Glassmorphism**: Modern UI with blurred backgrounds and subtle borders.
- **Video Player**: Responsive hero video with full controls.
- **App Grid**: 3-4 column layout optimized for mobile.
- **Smooth Navigation**: Smooth scrolling and floating home button.
- **Lightweight**: Built with pure HTML, CSS, and JS.

## Folder Structure
```
/
├── index.html      # Main entry point
├── style.css       # All styles and animations
├── script.js       # Interactive features and star generation
├── metadata.json   # App metadata
└── README.md       # Deployment instructions
```

## Deployment Instructions

### Deploying to GitHub
1. Create a new repository on GitHub.
2. Initialize your local project: `git init`.
3. Add files: `git add .`.
4. Commit: `git commit -m "Initial commit"`.
5. Link to GitHub: `git remote add origin <your-repo-url>`.
6. Push: `git push -u origin main`.

### Deploying to Vercel
1. Go to [Vercel](https://vercel.com).
2. Click **"New Project"**.
3. Import your GitHub repository.
4. Vercel will automatically detect the project as a "Static Site".
5. Click **"Deploy"**.
6. Your site will be live at a `.vercel.app` URL.
